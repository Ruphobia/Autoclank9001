# Features

A running tour of the tools `ac9` has wired up today. Each feature
below is something the chat pipeline can pick up and dispatch to on
its own — the LLM-based tool router (the newest of the bunch)
decides which one to hand a prompt to.

Everything on this page runs against local weights on local hardware.
No provider in the path, no telemetry off-box.
